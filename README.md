# Introduction
This repository compiles the finalized R Markdown scripts used for the analyses reported in 'Seasonal drought reroutes Amazon soil microbiomes, with greater susceptibility in terra firme than white-sand forests
' (Finck et al., 2026). doi: tba

# Data availability
All raw sequencing data files are deposited in the NCBI Sequence Read Archive (SRA) under the identifications PRJNA1425940. 

# Analysis files 
All analysis files (e.g. ASV tables, metadata etc.) are available upon reasonable request through the corresponding author.
